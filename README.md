# BMLD-Inf2-AppV0

Der Allergie-Tracker ist eine Web-App zur präzisen Erfassung und Analyse von Lebensmittelunverträglichkeiten. Die App hilft Nutzern dabei, Zusammenhänge zwischen ihrer Ernährung und körperlichen Symptomen objektiv zu identifizieren. Die Nutzer können ihre Mahlzeiten tracken und auch ihre Symptome in der Web-App festhalten. Schlussendlich können sie mögliche Korrelationen zwischen ihrer Ernährung und ihren allergischen Symptomen in einer Grafik herauslesen. Ausserdem können sie sich auf einer seperaten Seite durch die häufigsten Allergene klicken, mehr über diese erfahren und mögliche Gemeinsamkeiten feststellen.
