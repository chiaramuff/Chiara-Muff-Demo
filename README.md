# BMLD-Inf2-AppV0

Der Allergie-Tracker ist eine Web-App zur präzisen Erfassung und Analyse von Lebensmittelunverträglichkeiten. Die App hilft Nutzern dabei, Zusammenhänge zwischen ihrer Ernährung und körperlichen Symptomen objektiv zu identifizieren.
